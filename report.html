Getting started: Use Claude Code with a test-first prompt pattern. Ask it to write pytest tests for each agent module first, then implement until all tests pass.

Build a multi-agent review analysis pipeline with 5 agents. For EACH agent module: 1) First write a pytest test file that validates inputs, outputs, error handling, and edge cases (empty data, missing keys, API failures). 2) Then implement the module. 3) Run pytest after each module and fix until all tests pass before moving to the next agent. 4) Include an integration test that runs the full pipeline with synthetic data and asserts the final dashboard output is valid HTML with all expected sections. Do not move on from any module until its tests are green.
